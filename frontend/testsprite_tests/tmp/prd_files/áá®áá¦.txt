1. 🔌 데이터 연결 및 프록시 (최우선)
API 프록시: http://localhost:3000/api/kr/signals 호출 시 Flask 서버(:5001)로 정상 연결되는지 확인.
실데이터 연동: page.tsx의 하드코딩된 데이터를 fetch 로직으로 교체.
CORS 및 에러 핸들링: 서버 다운 시 대시보드가 하얗게 죽지 않고 "연결 끊김" 아이콘을 띄우는지 확인.
2. 🎨 UI/UX 및 글래스모피즘
다크모드 깊이감: 순수 블랙(#000) 배경 위에서 카드들이 적절한 투명도와 블러(Blur) 효과를 유지하는지.
폰트 렌더링: Mac 환경에서 San Francisco 폰트(system-ui)가 의도대로 두껍고 선명하게 나오는지.
스크롤 성능: GlassCard가 많아질 때 스크롤 시 버벅임(Jank)이 없는지 (Backdrop-filter는 성능 비용이 큼).
3. 📱 반응형 및 레이아웃
Dynamic Island 헤더: 모바일에서 상단 헤더가 화면을 가리지 않고 적절히 축소되는지.
Bento Grid: 창 크기를 줄였을 때 3열 -> 1열로 자연스럽게 카드들이 재배치(Masonry) 되는지.
